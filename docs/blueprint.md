# **App Name**: Vodafone Cash Automator

## Core Features:

- SMS to Google Sheets: Automatically forward Vodafone Cash SMS messages to Google Sheets using a Webhook triggered by a service outside of the application.
- Message Processing: Parse and extract key information from the SMS messages within the Google Sheet (operation type, amount, phone number, balance, cumulative total).
- Report Generation: Generate daily and monthly reports based on the processed data.
- Telegram Bot Integration: Automatically send the generated reports to a Telegram Bot.
- Subscription Management: Implement a free subscription system to control customer access (activate/deactivate) using Google Sheets.
- Customer Subscription Database: Database in the Google Sheet for managing the subscribers, with customer data, start date and end date.
- Summary generation: The app will be a tool to summarize Vodafone Cash transaction in simple terms, which is generated on a monthly and annual basis using a Generative AI service.

## Style Guidelines:

- Primary color: Dark teal (#008080) to convey trust and financial focus.
- Background color: Very light grey (#F0F0F0) to ensure readability and a clean interface.
- Accent color: Orange (#FFA500) for call-to-action buttons and important notifications.
- Headline font: 'Poppins' sans-serif for headlines, modern and easy to read.
- Body font: 'PT Sans' for the body, for a professional, legible look.
- Use simple, clean icons related to finance and data visualization.
- Subtle animations to confirm actions and improve user experience.
